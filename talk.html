The History of Computing

By Eric Moynihan

1812

Charles Babbage

Born in 1791, Babbage is known for inventing the digital computer.

The computers he created were quite different to modern computers. These computers were mechanical.

Computers back then were people. A common task these "computers" had was to create mathematical tables by hand. These mathematical tables were therfore expesnive to produce and sometimes had minor inaccuracies.

One of Babbage's first computers was the Difference Engine. This had instruction based processing and the control unit. This machine was able to tabulate (onto sheets) polynomial functions.

He only managed to produce a protype at the time however since funding fell through. All difference engines operational today are replicas.

Analytical Engine

This mechanical computer by Babbage was programmed using punch cards.

It was able to use it's previous output as an input to its next calculation and was in principle turing-complete

However just like his preivous project, this device was left unfinished due to innadequate funding

1854

Boolean Algebra

The fundamentals of modern computing stem back from the invention of boolean algebra.

George boole, the First Professor of Mathematics at UCC is responsible for this invention

What he proposed

He proposed that all mathematical operations can be represented using the following basic operations:

  • AND
  • OR
  • NOT

This abstraction paved the way for the implementation of processors that allow us to compute in the modern era

1943

WWII

The US Navy wanted to construct a more accurate torpedo for underwater conflict. They decided to make what would be know as an electromechanical computer.

Torpedo Data Computer

The computer was small enough to be loaded onto a torpedo. It tracked the position of the target and computed the angle that the torpedo had to go at.

1946

ENIAC

This is the first general purpose turing-complete computer

It consisted of 20x10 digit signed accumulators (result register), ring counters (made of vacuum tubes to store digits) and IO through IBM punch cards.

Yes it used base 10

  • It had the ability to even do floating point arithmetic with double precision (it was jerry rigged though).
  • It's original programmers consisted of Kay McNulty, Betty Jennings, Betty Snyder, Marlyn Meltzer, Fran Bilas, and Ruth Lichterman (all women)
  • It's first use case was to compute test simulations of the hydrogen bomb. Used 1 million cards for this. (this first large codebase)
  • Later revision of this became the EDVAC

Over the next few decades things got smaller

1952

IBM mainframes

First starting off with vacuum tubes and later PCBs and transistors, these computers powered businesses.

These would cost millions and would go on to run the financial industry to this day

Also IBM would sell decades long support contracts (lots of money)

COBOL

Grace Hopper, an American Computer Scientist invented FLOW-MATIC. This inspired COBOL which is one of the first main stream high level programming languages.

Designed to run on mainframes, it was marketed at businesses and still runs on mainframes to this day (particularily in fintech).

1970

PDP-11

Around the same time as microprocessors were taking off, the PDP-11 (equiped with a 16-bit minicomputer) was released.

It was poplular as it was sold for much less than a mainframe (selling 100'000s of models).

VAX

These were computers based off the PDP-11's instruction set but were 32-bit

Easly VAX's had backwards compatibility with PDP-11 programs and hence were called VAX-11's

1971

Unix

Unix was officially released as proprietary software at Bell Labs

It being a full OS stack with many cli utilities that influence computing today.

It was developed by some of the most influential people in computing. Dennis Ritchie and Ken Thompson

(was made before the POSIX standard)

1974

Intel 8080

One of the first microprocessors. The 2nd 8-bit microprocessor ever. It was significant since it's instruction set is what many modern Intel processors are based on (including x86)

1983

Apple

Apple releases the Apple Lisa; one of the first PC's with a Graphical User Interface. It had it's own office suite and had a mouse driven UI just like modern OS's

1984

Mac OS

Classic Mac OS was released along with the original Macintosh. It contained many of the features seen in the Lisa with a more intuitive UI

1985

Microsoft Windows

Windows is released! It contains many of the ideas that were shown in Mac OS (coincidentally?) but was sold to OEM's, undercutting Apple's Mac

1988

POSIX

The Portable Operating System Interface (POSIX) was a family of standards by the IEEE Computer Society. It defined the API of Operating Systems as well as shell utilities for compatibility with Unix.

In other words it tells you how to be Unix-like.

This lead to the creation of many of many POSIX compliant OS's such as MINIX with their micro-kernel or Linux with it's monolithic kernel

Now Windows is technically somewhat POSIX compliant since it contains a mostly unused POSIX subsystem in order to sell Windows to the US government

1991

Linux

The Linux kernel is released. It being a monolithic kernel with a GPL license that is.

Combined with the GNU project started a few years prior, this became a working operating system with UNIX like functionality (and a GUI a few years later)

2001

United States v. Microsoft Corp.

After years of outcompeting it's competitors through methods such as forcing it's OEM's to use Internet Explorer as it's default browser, the US brought them to court.

The result was to split Microsoft into one company producing Windows, and the other producing it's suite. Microsoft appealed and got that suite dropped for a lesser antitrust penalty.
